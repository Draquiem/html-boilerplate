This particular page is about links! There are three kinds of links:

  • Absolute Links, like to Mozilla Developer Network, which is a very good resource for web developers.

    • Most detailed way you can refer to a webresource. They start with the "scheme" (Typically http:// or https://), followed by the domain name of the website, then path of the target web page.

  • Relative Links, like to our extras page.

    • Root-relative links, like to the home page of our website, but those aren't useful to use right now.
